# Starforged AI Game Master - Quick Start Guide

Welcome, Spacer! This guide will get you playing in under 5 minutes.

## Starting the Game

### 1. Start the Server
```bash
# In your terminal
cd Datasworn
python main.py
```

### 2. Open the Game
Visit **http://localhost:5173** in your browser.

### 3. Create Your Character
Enter your character name and starting vow when prompted.

---

## Playing the Game

### Basic Controls

| Control | Action |
|---------|--------|
| **Type in the input box** | Describe what your character does |
| **Press Enter** | Submit your action |
| **Press R** | Open the dice roller |
| **Press F5** | Quick save |
| **Press F9** | Quick load |
| **Press ?** | Show all keyboard shortcuts |

### UI Elements

- **Left Panel**: Your character portrait, stats, and location image
- **Right Panel**: The narrative - where the story unfolds
- **Bottom Bar**: Quick actions for Saves, Recap, and Export

## Tips for New Players

1. **Be descriptive** - "I search the cargo hold for clues" works better than "I search"
2. **Embrace failure** - Misses create interesting stories!
3. **Use the recap** - If you forget where you were, the recap helps
4. **Save often** - The auto-save is great, but manual saves before big decisions help
5. **Experiment** - Try different approaches to situations

---

## Troubleshooting

### "Session not found" error
Restart the game - sessions reset when the server restarts.

### Images not loading
API rate limits may be in effect. The game still works without images.

### Game feels slow
The AI is thinking! Complex actions take longer to generate.
